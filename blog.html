1. Differences between local storage VS session storage

The Main difference between localStorage and sessionStorage is that localStorage data does not expire, whereas sessionStorage data is cleared when the page session ends

2. Differences between global scope VS block scope in javascript

A variable when declared inside the if or switch conditions or inside for or while loops or within a function, are accessible within that particular condition or loop.In simple word the variables declared inside the curly braces are called as within block scope whereas Global scoped variables can be accessed from anywhere in a JavaScript program. Variables declared with var , let and const are quite similar when declared outside a block. They all have Global Scope

3. How js event loop works

The event loop is the secret behind JavaScript's asynchronous programming.It monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it

4. In how many ways we can get undefined in js

There are many reasons to get undefined value , here are the followings:
* When we try to access a non-existent property of object or non-existing array element with index .
* If a function does not return anything
* If parameter value is not passed will give undefined
* If a variable declared but has not been assigned a value
* If return has nothing on the right side it will return undefined
* Delete an element from an array will give undefined
* Setting a variable value directly to undefined
